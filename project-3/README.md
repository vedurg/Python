# 🎓 Student Data Organizer

A simple **menu-driven Python application** that helps manage student records efficiently using Python collections such as **Lists, Dictionaries, Tuples, and Sets**.

This project allows users to add, view, update, and delete student information while maintaining a unique list of subjects offered.

---

## ✨ Features

* ➕ Add new student records
* 📋 Display all student information
* ✏️ Update existing student details
* 🗑️ Delete student records
* 📚 Display unique subjects offered
* 🚪 Exit the application

---

## 🛠️ Collections Used

| Collection | Purpose                                   |
| ---------- | ----------------------------------------- |
| List       | Stores all student records                |
| Dictionary | Stores individual student details         |
| Tuple      | Stores student ID reference               |
| Set        | Stores unique subjects without duplicates |

---

## 📂 Student Information Stored

* Student ID
* Name
* Age
* Grade
* Subjects

---

## ▶️ How to Run

1. Install Python 3.
2. Download or clone this repository.
3. Open a terminal in the project folder.
4. Run the program:

```bash
python Student_Data.py
```
