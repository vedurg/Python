# 👨‍💼 Employee Management System

A simple **Python Object-Oriented Programming (OOP)** project that demonstrates the core concepts of **Classes, Objects, Inheritance, Encapsulation, Constructors, Getter & Setter Methods, and Method Overriding** through an Employee Management System.

---

## ✨ Features

* 👤 Create an Employee
* 👨‍💼 Create a Manager
* 👨‍💻 Create a Developer
* 📋 Display Employee, Manager, or Developer details
* 🔒 Demonstrates encapsulation using protected attributes
* 🧬 Uses inheritance and method overriding
* 🖥️ Menu-driven console application

---

## 🛠️ OOP Concepts Used

* Classes & Objects
* Constructors (`__init__`)
* Inheritance
* Encapsulation
* Getter & Setter Methods
* Method Overriding
* Destructor (`__del__`)
* Loops and Conditional Statements

---

## ▶️ How to Run

* Run the Python file:

```bash
OOP.py
```

---

## 🎯 Learning Outcomes

* Understand the fundamentals of Object-Oriented Programming.
* Learn how inheritance improves code reusability.
* Practice encapsulation using protected attributes.
* Implement method overriding in derived classes.
* Build a menu-driven Python application.
