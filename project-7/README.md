# 🚀 Multi-Utility Toolkit (Python)

A simple Python project that demonstrates the use of **modules**, **packages**, and **built-in Python libraries** through a menu-driven application.

---

## 📌 Features

### 📅 Datetime & Time Operations

* Display current date and time
* Calculate difference between two dates
* Format date and time
* Stopwatch
* Countdown Timer

### 🧮 Mathematical Operations

* Calculate Factorial
* Compound Interest Calculator
* Trigonometric Calculations
* Area of Circle

### 🎲 Random Data Generation

* Generate Random Number
* Generate Random List
* Create Random Password
* Generate Random OTP

### 🆔 UUID Generator

* Generate Unique Identifiers (UUID)

### 📂 File Operations

* Create a File
* Write Data
* Read Data
* Append Data

### 🔍 Explore Module Attributes

* View available attributes using `dir()`
* Supports:

  * `datetime`
  * `math`
  * `random`
  * `uuid`
  * `file_module`
