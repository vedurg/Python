# Personal Data Collector

A simple Python project that collects user information and displays data types, memory addresses, and approximate birth year.

## Features
- Takes user input
- Uses `int()` and `float()` conversion
- Displays variable types with `type()`
- Displays memory addresses with `id()`
- Calculates birth year

## Technologies
- Python 3

## Run the Program

```bash
python main.py
```

## Example Output

```text
Enter your name: Rahul
Enter your age: 20

Your birth year is approximately: 2006
```

## Concepts Used
- Variables
- User Input
- Data Types
- Type Casting
- String Formatting
