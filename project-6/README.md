# 📖 Personal Journal Manager

A simple Python project that allows users to maintain a personal journal using file handling and object-oriented programming (OOP).

## ✨ Features

* Add a new journal entry
* View all saved entries
* Search entries using a keyword or date
* Delete all journal entries
* Automatically saves the current date and time with each entry

## 🛠️ Technologies Used

* Python 3
* File Handling
* OOP (Classes and Objects)
* `datetime` Module

## ▶️ How to Run

```bash
python File_Ope.py
```

## 📂 File Used

* `journal.txt` – Stores all journal entries.

## 🎯 Learning Outcomes

* File handling in Python
* Reading and writing text files
* Exception handling
* Working with classes and objects
* Using the `datetime` module
* Building a menu-driven application
