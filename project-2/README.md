# Pattern Generator and Number Analyzer

## Overview
This is a simple Python console-based project that can:

- Generate different star patterns
- Analyze numbers in a given range
- Calculate sum and average
- Check Even and Odd numbers

---

## Features

### Pattern Generator
- Right-angled Triangle
- Left-angled Triangle
- Full Triangle

### Number Analyzer
- Even/Odd Checker
- Sum of numbers
- Average calculation

---

## Technologies Used

- Python
- Match Case
- Loops
- Conditional Statements

---

## Concepts Used

- `while` loop
- `for` loop
- `range()`
- `match-case`
- Nested menu
- Arithmetic operations
- String multiplication
